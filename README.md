# Sandy Auto

Cross-platform automation with AI agents. Sandy Auto runs bots on your devices using Gemini, Mistral, Grok, or Claude to perform tasks on your behalf. Manage accounts, projects, and let AI live inside your device.

## Features

- Multi-device support: Android, iOS, Windows, macOS, Linux
- AI-driven automation without manual scripting
- Account linking and secure credential storage
- Project management dashboard
- Native performance with Rust acceleration

## Setup

1. Clone repository
2. Obtain API keys for AI providers
3. Build with Gradle:
   - Android: `./gradlew :androidApp:assembleDebug`
   - Desktop: `./gradlew :desktopApp:run`
   - iOS: Open `iosApp/iosApp.xcodeproj` in Xcode
